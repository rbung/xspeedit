## Prerequisites

 * Git installed
 * Node 8 installed
 
## Install it

```bash
git clone git@github.com:rbung/xspeedit.git
cd xspeedit
npm install -g
```

You should know have `xspeedit` command :)

## Play with it

 * Launch the CLI
```bash
xspeedit
```
